Finally, this is my code to solve the exercise:
```
from pibot.pibot 
import PiBot
import time

if __name__ == "__main__":

    robot = PiBot()   
    robot.quienSoy() 
    while True:
        ir = robot.leerIRSigueLineas()
       
        if ir == 0:
            robot.avanzar(0.1)
            time.sleep(0.01)
            robot.parar()
        elif ir == 1:
            robot.girarIzquierda(0.4)
            time.sleep(0.01)
            robot.parar()
        elif ir == 2:
            robot.girarDerecha(0.4)
            time.sleep(0.01)
            robot.parar()
        else:
            robot.retroceder(0.1)
            time.sleep(0.01)
            robot.parar()
 ```
